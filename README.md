# Freedom Travel OS v1.14.0

A premium offline-first travel PWA for the August 2026 Korea trip.

## This release
- Memories timeline with photos, captions, dates and favourites
- Trip Journey timeline from Singapore departure to return
- Explicit Save/Cancel editing for Flight and Hotel
- Readable form labels and protected unsaved changes

## Run locally
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```
