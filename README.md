# Freedom Travel OS — Version 1.0

Freedom Travel OS is a personal offline-first travel companion for the Korea / BIGBANG Goyang trip.

## Status

**Version:** 1.0.0  
**Release:** Version 1.0 Release Build  
**Milestone:** Feature complete, ready for real trip use and final personal QA.

## What is included

- Journey command centre
- Flight manager
- Hotel manager
- Explore planner
- Budget tracker
- Packing checklist
- Booking / price tracker
- Memories and notifications areas
- Light, dark, and system theme support
- Local storage persistence
- Windows helper scripts
- Production build in `dist/`

## Fast start on Windows

Extract this ZIP to a short path to avoid Windows path-length issues, for example:

```powershell
C:\FTOS\v1
```

Then run one of these:

```powershell
npm install
npm run dev
```

Or double-click:

```text
RUN_DEV_WINDOWS.bat
```

For PowerShell:

```powershell
.\RUN_DEV_POWERSHELL.ps1
```

## Production build

```powershell
npm install
npm run build
```

The production output is created in `dist/`.

## Development rule after v1.0

Version 1.0 is feature complete. New ideas should move to the Version 1.1 roadmap unless they are critical for Korea trip readiness.
