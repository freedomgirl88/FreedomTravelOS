# Freedom Travel OS v1.1 Quick Start

## Windows preview
1. Extract the ZIP to a short path such as `C:\FTOS11`.
2. Double-click `RUN_DEV_WINDOWS.bat`, or run `npm install` then `npm run dev`.
3. Open the local address shown by Vite.

## GitHub Pages update
1. Replace the repository source with this release.
2. Run `npm install` and `npm run build`.
3. Deploy the `dist` folder using the same working GitHub Pages workflow already used for the project.
4. After deployment, fully close and reopen the installed PWA.
5. If old flight/hotel data remains, use Settings → Reset Data once.

## Flight alert setup
1. Open More → Notifications.
2. Tap **Enable live alerts** and allow notifications.
3. Tap **Add backup calendar reminder** and add the event to the phone calendar.
4. Keep a phone alarm as a final backup for the 11:00 departure on 26 Aug.
