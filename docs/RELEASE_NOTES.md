# Release Notes — Freedom Travel OS v1.0

Freedom Travel OS v1.0 is the first complete release build. It is intended to be used as the main personal travel companion for the August 2026 Korea / BIGBANG trip.

## Highlights
- Premium light and dark themes.
- Journey command centre with readiness, focus, quick actions, and assistant reminders.
- Flight, hotel, budget, packing, explore, and booking workflows.
- Booking page uses a clear inline **Add Price Check** action instead of overlapping floating controls.
- Budget page keeps the primary **Add Expense** action in the hero card.
- Local storage saves trip progress on the device.
- Windows helper scripts added for easier launching without PowerShell `&&` issues.

## Release rule
No new major features should be added before the Korea trip unless they are critical. Version 1.1 should be based on real use during travel.
