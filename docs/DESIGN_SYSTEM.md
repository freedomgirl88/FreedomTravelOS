# Design System

## Identity
Premium mobile-first travel app with two official themes:
- Light: soft blue / white travel OS.
- Dark: deep navy travel cockpit inspired by the user's reference screenshot.

## Theme Modes
- Light
- Dark
- System

Theme preference is stored locally. System mode follows the device appearance.

## Core Tokens
- Primary: #5b7cfa
- Primary Gradient: #5b7cfa → #7c5cff
- Light Background: #f6f8fd
- Light Card: rgba(255,255,255,.92)
- Dark Background: #071121 / #09111f
- Dark Card: rgba(31,42,58,.94)
- Border: #dce4f4 light / rgba(151,164,184,.22) dark
- Muted Text: #6b7890 light / #a9b5c9 dark

## Rules
- Important info visible within 3 seconds.
- Use bottom sheets instead of long visible forms.
- Keep cards aligned and readable.
- Avoid unnecessary scrolling.
- Use toast feedback instead of disruptive alerts where possible.
- Preserve safe bottom spacing so navigation never hides content.
