# Freedom Travel OS Project

Current version: **1.0.0**

Status: **Version 1.0 Release Build**

## Mission
Freedom Travel OS helps manage the user's Korea trip through one premium, offline-first travel companion.

## Core modules
- Journey
- Flight
- Hotel
- Explore
- Budget
- Packing
- Booking History
- Memories
- Notifications
- More / Settings

## Design direction
- Premium light blue / white mode.
- Premium dark navy mode inspired by the approved reference style.
- Rounded cards, glass navigation, calm gradients, readable typography.

## Post-release rule
After v1.0, avoid speculative feature additions. Improvements should come from actual use before and during the Korea trip.
