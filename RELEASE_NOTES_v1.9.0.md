# Freedom Travel OS v1.9.0 — Airport Journey 3.0

- Rebuilt airport timing around one Journey Engine.
- Added separate Normal Planning and Emergency modes.
- In Normal Planning, flight/airport timing is the anchor and AREX/Taxi calculate different hotel departure times.
- In Emergency Mode, an exact hotel departure is intentionally pinned and airport arrival changes by transport.
- Added visible leave-time previews on both transport cards.
- Changing flight departure, airport arrival target, buffer, or transport recalculates the full timeline.
- Added one-time migration out of the old silently pinned manual mode.
